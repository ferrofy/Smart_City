Follow These Steps To Prepare The Complete Hardware And Software Demo Scenario.

## 🛠️ Prerequisites

* Python Version Supported On Target System.
* Arduino IDE Configured For Proper Board Pluggins.
* System Audio Or Data Input Permissions Granted.

## 🏃‍♂️ Running The Stack

### 1. Arduino Configuration

* Upload The Pre-Written Arduino Code File To The Target Board.
* Ensure Sensors And Servo Motors Are Wired Correctly As Per Architecture Diagrams.

### 2. Microcontroller Setup

* Set Up Your Local Wireless Credentials In The Module Configurations.
* Connect Module And Ensure Successful Dashboard Ping.

### 3. Deep Learning Base

* Install Requirements Through Standard Package Management Options.
* Ensure External Weights Are Located Inside The Safe Model Directory.

### 4. Boot Up System

* Run The Primary Execution Script Using System Terminal.
* Observe The Processing Logic Activating Within Optimal Timeframes.
* Observe AI Classifications Displaying Over Identified Urban Data.
